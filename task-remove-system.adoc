= 从NetApp Console中删除Azure NetApp Files
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以从NetApp Console中删除Azure NetApp Files系统。删除Azure NetApp Files系统不会删除你的Azure NetApp Files帐户、容量池或卷。您可以随时将Azure NetApp Files添加回控制台。

.步骤
. 打开Azure NetApp Files系统。
. 在页面右上角，选择三个点 `...`然后*删除Azure NetApp Files*。


image:screenshot-remove-system.png["显示删除Azure NetApp Files系统选项的屏幕截图。"]

. 在弹出的对话框中，选择*删除*以确认您要删除该系统。
